## Install

```sh
npm i -g dev-command
# or
yarn global add dev-command
```

## Usage

### Init boilerplates

The `init` command will generate flexible config files or scripts and install all npm dependencies silently.

It's so simple to add good linter for your exists project.

```sh
dev init
```
